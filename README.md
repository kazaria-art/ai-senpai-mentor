# AI先輩メンター

新入社員向け対話型社員教育支援Webサイト

## ファイル構成

```
ai-senpai-mentor/
├── index.html       ← メインファイル
├── matsuoka.png     ← 松岡先輩の画像
├── nozomi.png       ← のぞみ先輩の画像
├── reizei.png       ← 冷泉先輩の画像
└── README.md        ← このファイル
```

## GitHub Pagesへの公開手順

1. GitHubで新しいリポジトリを作成（例：`ai-senpai-mentor`）
2. このフォルダ内の **4ファイルすべて** をリポジトリにアップロード
3. リポジトリの「Settings」→「Pages」を開く
4. Branch を `main`、フォルダを `/ (root)` に設定して「Save」
5. 数分後に `https://ユーザー名.github.io/ai-senpai-mentor/` で公開される

## 注意事項

- `index.html` と画像ファイル（png）は **必ず同じフォルダに置いてください**
- 画像ファイルのファイル名を変えると表示されなくなります
